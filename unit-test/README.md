## Mô tả
Bài tập thực hành kiểm thử đơn vị bằng JUnit cho chương trình phân tích điểm số học sinh.

## Chức năng
- Đếm số học sinh đạt loại Giỏi (>= 8.0)
- Tính điểm trung bình hợp lệ (0 - 10)
- Bỏ qua dữ liệu sai (<0 hoặc >10)
